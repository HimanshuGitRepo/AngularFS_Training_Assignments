Angular Form Validations using Model Driven

Car Registration Form Owner Name : Owner Name is required

Contact Number : Contact Number is required and should be of atleast 10 digits.

E-mail : Enter proper e-mail id

Vehicle Registration Number : Enter Proper Vehicle Registration Number.

Address : Address is required

Model : Model should be ten digits.

Colour : Colour is required.

Year : Year should be in between 2012 to 2022.
